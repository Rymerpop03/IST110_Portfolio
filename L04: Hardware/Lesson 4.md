# Hardware Discussion
Key Components: The main components of a computer are: GPU, CPU, RAM, PSU, SSD, and Motherboard. The GPU, graphics processing unit, is the component in charge of processing tasks/graphics on the computer. The CPU, central processing unit, is the brain of the computer, it executes instructions from computer programs by performing calculations. The RAM, memory, defines the amount of processes the computer can run at a time. The PSU is power supply unit which supplies electricity to the rest of the computer's components. Finally is the motherboard, the piece that holds all the components in together, it allows them to interact with one another.
